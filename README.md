# 🛒 ShopKart - E-commerce Website

## 📌 Project Description
This is a Django-based E-commerce website where users can view products, add items to cart, and explore an online shopping experience.

## 🚀 Features
- Product listing page
- Add to cart functionality
- Simple UI with HTML, CSS
- Django backend
- SQLite database (local use)

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript
- Python (Django)
- SQLite

## 📂 Project Structure
- shopkart/ → Main project
- store/ → App (products, views)
- templates/ → HTML pages
- static/ → CSS files

1. Clone the repo
2. Install dependencies
3. Run:
